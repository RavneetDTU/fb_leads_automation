# Mets Leads

Lead management platform with Google Calendar integration.

## Setup

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Configure environment**
   ```bash
   cp .env.example .env
   ```
   Add your Google Client ID to `.env`

3. **Run development server**
   ```bash
   npm run dev
   ```

## Build

```bash
npm run build
```

## Tech Stack

- React + Vite
- Tailwind CSS
- React Router
- Google Calendar API
